Aplication's role is to draw a random number from config file.

<-- config.txt -->
line:
	1st: minimal number
	2st: maximal number
	3rd: Text for case if all numbers were chosen
		-> number draw will be repeated in next application open
	4th: Regular text to drawn number
		-> number chosen from range between minimal and maximal values

<-- numbers.txt -->
store already drawn numbers
